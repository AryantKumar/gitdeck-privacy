Privacy Policy for GitDeck

Last updated: February 13, 2026

GitDeck ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how our GitDeck mobile application (the "App") collects, uses, and discloses your information.

1. Information We Collect

1.1 GitHub Data

GitDeck is a third-party client for GitHub. When you log in with your GitHub account, we access the following data via the GitHub API:

  • User Profile: Username, avatar, and public profile information.
  • Repository Data: Code, issues, pull requests, and notifications for repositories you have access to.
  • Activity: Your interactions (e.g., starring repos, commenting on issues).

1.2 Device Information

We may collect minimal device information (e.g., operating system version) solely for the purpose of debugging crashes or improving App performance.

2. How We Use Your Information

We use the collected information for the following purposes:

  • Functionality: To display your GitHub dashboard, manage issues/PRs, and provide the core features of the App.
  • Offline Access: We store a local copy of your data on your device to allow the App to function without an internet connection.
  • Improvements: To analyze crash reports and improve the stability of the App.

3. Data Storage and Security

  • Local Storage: All your GitHub data (repositories, issues, etc.) is stored locally on your device using a secure database. We do not store your code or personal data on our own external servers.
  • Authentication: We use industry-standard OAuth 2.0 to authenticate with GitHub. Your GitHub password is never seen or stored by GitDeck. Access tokens are stored securely using your device's native secure storage (Keychain on iOS, Keystore on Android).

4. Third-Party Services

The App interacts with the following third-party services:

  • GitHub API: To fetch and update your data. Their privacy policy can be viewed at GitHub Privacy Statement.
  • Google Play Services / Apple App Store: For app distribution and crash reporting (if enabled).

5. Your Rights and Control

  • Revoke Access: You can revoke GitDeck's access to your GitHub account at any time by visiting your GitHub Settings > Applications.
  • Delete Data: You can delete all locally stored data by logging out of the App or uninstalling it.

6. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.
