# GitDeck Privacy Policy

This repository hosts the **Privacy Policy** for the **GitDeck** mobile application.

## Overview
GitDeck is a third-party GitHub client. This privacy policy outlines how we handle user data, adhering to strict privacy standards. We do not store personal data on external servers; everything is kept locally on your device.

You can view the full privacy policy in the [index.html](index.html) file of this repository, or hosted live at:
